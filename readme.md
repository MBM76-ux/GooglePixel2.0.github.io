## 📌 Project Overview

Google Pixel Clone is a fully interactive, multi-page web application that simulates a smartphone product showcase website. It was built as Assignment 02 for the Web Engineering course at The University of Lahore (BSCS Spring 2026).

The project transforms a static website into a dynamic, data-driven application featuring full CRUD operations, advanced array/object/string methods, search & filter functionality, and a persistent dark/light theme toggle.

---

## ✨ Key Features

- ✅ **5 Web Pages** — Home, About, Contact, Sign In, Sign Up
- ✅ **CRUD App** — Add, Read, Update, Delete phone devices using Array of Objects + DOM
- ✅ **Search & Filter** — 6+ filters using filter(), map(), sort(), reduce(), includes(), find()
- ✅ **Dark / Light Mode** — Theme toggle with localStorage persistence across all pages
- ✅ **Object Methods** — Object.keys, values, entries, assign, freeze, delete, hasOwnProperty, fromEntries, spread
- ✅ **10+ String Methods** — trim, toUpperCase, toLowerCase, includes, startsWith, endsWith, replace, split, slice, indexOf, padStart
- ✅ **Control Structures** — If-Else, For Loop, While Loop, Nested Loop + Conditions
- ✅ **Form Validation** — Sign In and Sign Up with regex-based JS validation
- ✅ **ES6 Standards** — Arrow functions, template literals, destructuring, spread operator

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/google-pixel-clone.git
   ```

2. Open in VS Code:
   ```bash
   cd google-pixel-clone
   code .
   ```

3. Install the **Live Server** extension in VS Code.

4. Right-click on `index.html` → **Open with Live Server**

5. The site will open at `http://localhost:5501`
